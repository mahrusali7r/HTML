Inline Elements

An inline element does not start on a new line. An inline element only take up as much width as necessary.

List of Inline Elements

  1. a
  2. strong
  3. em
  4. span
  5. img
  6. audio
  7. video
  8. code

Block Elements

Block elements in HTML are elements that typically start on a new line and extend to the full width available. They create "blocks" of content.

List of Block Elements

  1. p
  2. header
  3. h1 to h6
  4. article
  5. section
  6. aside
  7. footer
  8. table
  9. form
  10. pre
  11. div
