HTML Entity

HTML Entities are speical strings of characters that start with a ampersand (&) and end with a semicolon (:). These entities represent specific symbols and can be indentified by their entity name or number.

There are three different ways to format a character entity in HTML.

  1. By Using the Character Code
  2. By Using a Unicode value
  3. By Using a Number
