1. DOCTYPE Declaration

<!DOCTYPE html>

Purpose: This line tells the web browser that the document is an HTML5 document. It's important because it ensures that the browser renders the page correctly.

2. HTML Element

<html lang="en">

Purpose: The <html> tag is the root element of the document, enclosing all other HTML content. Attributes: lang="en": Specifies that the language of the document is English.

3. Head Section

<head>

Purpose: The <head> section contains meta-information about the HTML document, such as its title, character encoding, linked stylesheets, and scripts.

a. Meta Tags

<meta charset="UTF-8">

Purpose: Specifies the character encoding for the document, which is UTF-8. This ensures that the document can display a wide range of characters from different languages.

b. Link to CSS

<link rel="stylesheet" href="style.css">

Purpose: This tag links to an external CSS file (`style.css`) that contains the styles (like colors, fonts, layout) for the webpage.

c. Title Tag

<title>World General Library </title>

Purpose: This tag links to an external CSS file (`style.css`) that contains the styles (like colors, fonts, layout) for the webpage.

4. Body Section

<body>

Purpose: The <body> section contains the actual content of the webpage, including text, images, links, and more. All visible content that appears on the page goes inside this tag.

5. Script Tag

<script src="script.js"> </script>

Purpose: Links to an external JavaScript file (`script.js`). JavaScript is used to add interactive features and dynamic content to the webpage.
