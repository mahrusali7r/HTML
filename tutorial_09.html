HTML IDs & Classes

HTML IDs

One ID is asigned to only one element and an element has only one ID. ID must be unique. It is represented with a "#" sign.

Examples

id="Alfa", id="PAK"

<p id="alfa">, <h1 id="pk">

<span id="beta">, <div id="pak">

HTML Classes

One class is asigned to multiple elements and an element has multiple classes. It is represented with a "." sign.

Examples

class="A", class="P"

<p class="gamma">, <h5 class="tera">

<span class="gamma">, <div class="container">
